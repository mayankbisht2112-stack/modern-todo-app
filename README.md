# Modern To-Do App

A responsive and interactive To-Do List web application built using HTML, CSS, and JavaScript.

## Features
- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Progress bar tracking
- Confetti animation on task completion
- LocalStorage support
- Empty state image

## Technologies Used
- HTML5
- CSS3
- JavaScript
- LocalStorage API
- tsParticles Confetti

## How It Works
- Tasks are dynamically created using DOM manipulation.
- Completed tasks update the progress bar.
- Tasks are saved in localStorage so data persists after refresh.
